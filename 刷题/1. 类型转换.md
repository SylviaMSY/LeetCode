## 类型间转换

### 1 String 与 int

> String 转 int

``` java
int i = Integer.parseInt(str); //Integer.parseInt()得到基本数据类型int,无法处理非数值
int i = Integer.valueOf(str).intValue(); //Integer.valueOf()得到Integer,无法处理非数值
```

> int 转 String

``` java
String s = String.valueOf(i);
String s = Integer.toString(i);
String s = "" + i;
```
