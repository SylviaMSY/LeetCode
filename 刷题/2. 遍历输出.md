## 遍历输出

### 1 Map

> 利用iterator迭代器

``` java
// .1. 使用Map中的entrySet方法


// 2. 分别遍历key、value
Map<Object, Object> map = new HashMap<Object, Object>();
for(Object key : map.keySet){	//只能遍历key
	System.out.print("Key = "+key);
}
for(Object value : map.values){  //只能遍历value
	System.out.print("Value = "+value);
}
```
